Introdução ao Sistema

Bem-vindo ao Sistema TEC-MAQLI!
Este manual completo irá guiá-lo através de todas as funcionalidades do sistema de gestão de clientes e máquinas industriais.

Sobre o Sistema

O Sistema TEC-MAQLI LTDA é uma solução integrada para gerenciamento de clientes, máquinas industriais e peças de reposição. Desenvolvido para oficinas mecânicas industriais, oferece:

Gestão Completa de Clientes

Cadastro, edição e visualização detalhada de informações dos clientes

Controle de Máquinas

Registro e acompanhamento de máquinas industriais por cliente

Gestão de Peças

Cadastro e controle de peças de reposição para cada máquina

Busca Avançada

Localização rápida de clientes, máquinas e peças

Campos Obrigatórios:

Passo a Passo do Cadastro

1 Acessar Formulário

Na tela principal, toque no botão "Adicionar Cliente"

Excluir Cliente

Atenção:
A exclusão de um cliente removerá também todas as máquinas e peças associadas. Esta ação não pode ser desfeita.
1 Acessar Detalhes do Cliente

Na tela inicial, toque no cliente que deseja excluir

2 Localizar Botão de Exclusão

Na seção "Ações", toque no botão "Excluir Cliente"

O botão tem ícone de lixeira e cor vermelha

3 Confirmar Exclusão

Uma mensagem de confirmação será exibida

Toque em "Excluir" para confirmar ou "Cancelar" para abortar

4 Resultado

Em caso de sucesso, você será redirecionado para a tela inicial

Uma mensagem confirmando a exclusão será exibida

1 Acessar Lista de Máquinas

Dicas e Soluções

Esta seção contém dicas úteis e soluções para problemas comuns no sistema.

Perguntas Frequentes

Não encontro um cliente

Verifique se você está usando o termo correto na busca. O sistema pesquisa em nome, telefone e endereço simultaneamente.

Não consigo salvar um cliente

Certifique-se que todos os campos obrigatórios estão preenchidos corretamente. O sistema mostrará mensagens de erro específicas.

Formato de telefone inválido

O sistema formata automaticamente para (XX) XXXXX-XXXX. Digite apenas os números, o formato será aplicado automaticamente.

Email inválido

Se informar um email, ele deve conter um formato válido (ex: usuario@provedor.com). O email é opcional.

Mensagens de Erro Comuns

"Nome é obrigatório"

O campo Nome da Empresa não pode ficar vazio. Informe um nome válido para o cliente.

"Telefone é obrigatório"

O campo Telefone deve ser preenchido com um número válido no formato (XX) XXXXX-XXXX.

"Endereço é obrigatório"

O campo Endereço não pode ficar vazio. Informe um endereço válido para o cliente.

"Formato de email inválido"

O email informado não está em um formato válido. Verifique se contém "@" e um domínio correto.

Dicas de Produtividade

Busca Rápida

Use a barra de pesquisa na tela inicial para encontrar clientes rapidamente, sem precisar rolar a lista.

Cadastro Completo

Preencha todos os campos possíveis ao cadastrar um cliente para ter informações completas disponíveis.

Histórico de Atualizações

Na tela de detalhes, veja quando o cliente foi cadastrado e sua última atualização na seção "Informações do Sistema".

Design Responsivo

O sistema foi projetado para funcionar bem tanto em smartphones quanto em tablets.
